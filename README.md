# react-reflux示例
=============
#### 该项目包含：
  * react-router 的配置与使用;
  * react 搭配 reflux 进行数据流管理;
  * webpack@2.2.0 配置与使用;
  * json-server 模拟请求.

### 安装依赖
    npm install

### JSON-server
    1.    npm run json
    2.    http://127.0.0.1:3000/

### 开发运行
    1.    npm start
    2.    http://127.0.0.1:5050/

### Express 运行
    1.    npm run release
    2.    node server.js
    3.    http://127.0.0.1:3001/
